# AO3 Comment Previewer App

A standalone tool for previewing AO3 comment HTML before posting. Paste rich
text or raw HTML into the editor and see a sanitized live preview of exactly
how it will render as an AO3 comment.

Extracted from `uc41-html-formatted-chapters` (now `disco-elysium-css-formatter`),
where it originally lived alongside an unrelated chapter-formatting tool.

## Development

```
npm install
npm start      # build + serve locally
npm test       # run unit tests
npm run deploy # publish to gh-pages
```
